# NLP Utility (NER, Language Detection, Sentiment Analysis)

## A simple command-line NLP application built using Python + OOP and the NLPCloud API.
## It allows users to register, log in, and run NLP tasks:

### Features

- User Registration & Login System

- Named Entity Recognition (NER)

- Language Detection

- Sentiment Analysis

- Built using Python OOP, private methods, class structuring

- Integrates real NLP Cloud APIs for inference

### Stack

- Python 3
- NLP Cloud API
- OOP Principles

### Usage

1. Run the script
```
python project_1_nlp.
```
2. Register new users  
3. Login using stored credentials  
4. In-memory mini-database using Python dictionaries

### NLP Functionalities
1. **Named Entity Recognition (NER)**  
   - Extracts entities from the paragraph  
   - Allows searching for a specific entity  

2. **Language Detection**  
   - Detects the language of a given text  

3. **Sentiment Analysis**  
   - Returns positive/negative sentiment from the text
